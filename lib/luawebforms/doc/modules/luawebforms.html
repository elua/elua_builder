Module luawebforms

LuaWebForms is a Lua library developed to create and manipulate HTML forms. Some methods were based on Ruby on Rails Form Tag Helper, but new features were included to manipulate better "select", "radiobox" and "checkbox" tags.

Release: $Id: luaWebForms.lua,v 1.0 beta 2008/07/02 15:17:00 Vagner Nascimento Exp $

check_box (arg, values, checked_values, spacer, left_labels)
Returns a string with a group of check boxs tags.

Parameters

  • arg: Lua table with check box parameters
  • values: Accepts a table in the format: {{value="apple",label="Apple"},{value="orange", label="Orange"},{value="grape", label="Grapes"}}
  • checked_values: Accepts a table in the format: {"apple","grape"}
  • spacer: Puts the string sent after or before the check box. The default value is a " "(space)
  • left_labels: If the value is true, the labels are positioned before the check box. The default is the right side

Usage:

obj.check_box({name="fruits"},{{value="ap",label="Apple"},{value="or", label="Orange"}}, {"or","ap"})
