Module sool

SOOL - Simple Object Oriented lib for Lua. Based on several OO implementations for Lua available on the web, SOOL implements instantiation, static classes, multiple inheritance and object type verification. SOOL does not intend to be a complete implementation of OO for Lua, instead it intends to be simple and short.

sool.isA (objectOrClass, superclass)
Verifies if the object is an instance of the base class or if the class inherits from the base class.

Parameters

  • objectOrClass: object or class being verified
  • superclass: supposed parent class

Usage:

 print(sool.isA(object1, Class1))
 print(sool.isA(Class1, Class2))

Return value:

true if object is an instance of the base class or if the class inherits from the base class. Otherwise, returns false.
sool.package ()
Creates a new package. Should be applied to a module on its creation. Usually, a package is used to group classes.

Usage:

 module ("display", sool.package())
 
 Menu = require "display.Menu"
